# Software Journey - Smart Task Assistant

## Introducción

El presente Software Journey documenta el proceso de desarrollo del proyecto Smart Task Assistant, una aplicación diseñada para la gestión inteligente de tareas mediante una arquitectura basada en frontend y backend.

El proyecto fue desarrollado utilizando un enfoque de co-creación entre desarrollador y herramientas de inteligencia artificial, apoyándose en Claude Code para la generación de estructuras iniciales, planificación de funcionalidades y revisión arquitectónica.

La documentación se encuentra organizada en tres secciones principales:

1. La Bala Trazadora (Tracer Bullet) y el Enrutamiento de las Skills.
2. Anatomía de la Complejidad (Deep Modules vs Shallow Modules).
3. El Veredicto Retrospectivo de los Sub-Agentes.

Cada sección analiza el desarrollo del sistema utilizando conceptos de ingeniería de software moderna basados en la obra "A Philosophy of Software Design" de John Ousterhout.
